# Mahesh Yadav — Personal Portfolio 🚀

A clean, modern, dark-themed personal portfolio website built with pure HTML, CSS, and Vanilla JavaScript. Showcasing my skills, projects, and contact information.

🔗 **Live Site:** [maheshyadavpersonalportfolio.netlify.app](https://maheshyadavpersonalportfolio.netlify.app)

---

## ✨ Features

- **Responsive Design** — Works on all screen sizes (mobile, tablet, desktop)
- **Smooth Animations** — Scroll-reveal animations and hover effects
- **Typewriter Effect** — Animated role text in the hero section
- **Project Filter** — Filter projects by category (Frontend, Fullstack, Backend)
- **Contact Form** — Integrated contact form with validation
- **Dark Editorial Theme** — Professional dark UI with electric purple accent
- **Scroll to Top** — Smooth scroll-to-top button
- **Mobile Menu** — Hamburger menu for mobile navigation

---

## 🚀 Getting Started

### View Locally
Just open `index.html` in your browser — no build step or server needed.

### Deploy Changes
```bash
git add .
git commit -m "your update message"
git push
```
Netlify auto-deploys on every push to `main`.
